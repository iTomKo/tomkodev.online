OTTO - motd.yml

Updated for Otto version: 1.0.1.4

Go back to the main page

In this config you specify server motds.

There also is an option to select, whether the motds should be randomized (their order) or go in a linear loop.


Each additional motd is specified by "-" at the beginning of the line, with padding in front of it.

TIP: using this site you can easily create MOTD gradients!
